# 创作简报：我重生后靠签到系统无敌了

## 题材
玄幻 · 重生 · 系统流

## 核心设定
主角林渊，前世是天玄大陆普通修士，被未婚妻和宗门天才联手陷害，死于禁地。重生回到 16 岁觉醒当天，同时获得「每日签到系统」——每天签到可获得随机奖励（功法、丹药、神兵、灵宠、属性点等）。

## 世界观
- **天玄大陆**：修仙世界，分五大洲，宗门林立
- **修炼体系**：炼气 → 筑基 → 金丹 → 元婴 → 化神 → 合体 → 渡劫 → 大乘 → 飞升
- **资源稀缺**：高级功法和丹药被大宗门垄断，散修生存艰难

## 主角设定
- **林渊**：前世窝囊废，重生后利用签到系统 + 前世记忆双挂碾压
- **性格**：表面低调隐忍，实际上睚眦必报，打脸干脆利落
- **金手指**：签到系统（每日一次，地点不同奖励不同）

## 女主
- **苏清雪**：冰属性天才，表面冷淡，实则外冷内热
- **初期不在意男主** → 后被实力震惊 → 暗生情愫

## 反派
- **陈天行**：宗门天才，前世的仇人之一，装正派实际阴狠
- **赵婉儿**：前世未婚妻，拜金攀附，看不起主角

## 爽点节奏
1. **开局打脸**：觉醒仪式被嘲讽 → 签到获得顶级功法 → 震惊全场
2. **宗门大比**：隐藏实力 → 最后一鸣惊人 → 夺冠
3. **秘境夺宝**：利用前世记忆抢机缘 → 截胡反派的奇遇
4. **退婚反转**：赵婉儿来退婚 → 主角主动解除 → 她后悔莫及
5. **势力扩张**：建立自己的势力 → 招募天才 → 对抗大宗门

## 写作要求
- 节奏快，每章必须有冲突或反转
- 打脸要爽，不要拖泥带水
- 签到奖励要有惊喜感，但不能太离谱（前期适度）
- 留伏笔：禁地的秘密、系统的来历、前世的更多仇人
- 字数：每章 2000-3000 字
